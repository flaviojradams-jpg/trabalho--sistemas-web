# Petcare - Projeto de Fundamentos de Sistemas Web

Este projeto foi desenvolvido como parte da disciplina de Fundamentos de Sistemas Web, com o objetivo de criar um sistema web para um petshop, apresentando produtos, serviços e funcionalidades de cadastro de clientes e pets.

O sistema evoluiu ao longo das fases do projeto, partindo de uma estrutura básica em HTML para uma aplicação mais organizada, estilizada e interativa.

# Objetivo

Desenvolver um site de petshop que permita:

- Visualizar produtos organizados por categorias;
- Consultar serviços oferecidos;
- Realizar cadastro de clientes e pets;
- Agendar serviços de forma simples e intuitiva.

---

# Funcionalidades implementadas

- Página inicial com:
  - Cabeçalho  com barra de navegação;
  - Listagem de produtos (com imagens, descrição e preço);
  - Seção de serviços;
  - Rodapé com informações de contato;

- Sistema de navegação entre páginas (Home e Cadastro);

- Formulário de cadastro contendo:
  - Dados do cliente (nome, CPF, e-mail, telefone, endereço e sexo);
  - Dados do pet (nome, raça e idade);
  - Escolha do serviço (banho ou tosa);
  - Método de atendimento (tele-busca ou atendimento no local);
  - Agendamento (data e hora);
  - Validação básica com JavaScript (mensagem ao enviar formulário);

# Tecnologias utilizadas

- HTML5 → estrutura do site;
- CSS / Bootstrap → estilização e layout responsivo;
- JavaScript → interatividade e validação do formulário;
- GitHub Pages → hospedagem do site.

## Melhorias da fase 2

- Aplicação do Bootstrap para melhorar o layout e responsividade;
- Criação de cards para exibição dos produtos;
- Implementação de navbar responsiva;
- Padronização das imagens e layout visual;
- Criação de formulário completo de cadastro;
- Uso de JavaScript para interação com o usuário;
- Melhor organização do código e estrutura das páginas.

# Acessibilidade

O projeto contempla práticas básicas de acessibilidade, como:

- Uso do atributo `alt` em imagens;
- Associação de `label` com inputs em formulários;
- Estrutura semântica com `header`, `main`, `section` e `footer`.
