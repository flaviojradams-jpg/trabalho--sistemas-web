#Petcare - projeto de fundamentos de sistemas web

Este projeto de sistema web foi desenvolvido como parte de um trabalho acadêmico, e o objetivo é criar um site de petshop que apresente os seus serviços, informações de contato e afins
Atualmente este projeto se encontra em seu estágio inicial

#Tecnologias implementadas:
De acordo com o que a fase 1 do projeto pedia o sistema possui:
* página inicial com cabeçalho, corpo e rodapé; 
* catálogo com pelo menos 2 produtos, contendo descrições e fotos, de 3 categorias distintas (alimentos, acessórios e higiene). Além disso também constam os serviços fornecidos pelo petshop;
* Rodapé com as informações do estabalecimento, além do nome de quem o desenvolveu.

* Tecnologias utilizadas (por enquanto)
* HTML5 (criação da página)
* GithubPages (hospedagem)
